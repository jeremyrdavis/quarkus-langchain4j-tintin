# Product Mission

## Problem

Developers need a comprehensive, hands-on demo that showcases the full capabilities of the Quarkus Langchain4j extension — from basic AI services to advanced features like RAG, function calling, guardrails, image processing, and real-time chat. Existing examples tend to show features in isolation, making it hard to understand how they work together in a real application.

## Target Users

Java developers and architects evaluating Quarkus for AI-enabled applications, conference attendees, and workshop participants who want to see what's possible with Quarkus + LLM integration.

## Solution

A Tintin-themed demo application (inspired by the Belgian comic series by Hergé) that showcases nearly every Quarkus Langchain4j feature in a single, cohesive, fun application. The Tintin theme provides engaging domain content — character analysis, adventure classification, trivia, and RAG over a real Tintin book — making the demo memorable and entertaining while demonstrating production-ready patterns.
